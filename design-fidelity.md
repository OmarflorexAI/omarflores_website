# Design Fidelity

Reference: https://eriktaveras.com/

## Match These Patterns

- Dark editorial engineering aesthetic: near-black backgrounds, monospace section labels, green+gold accents
- Section labels use `// Comment` style: `<span class="font-mono text-sm text-brand-emerald uppercase tracking-widest">// Section Name</span>`
- Status indicator with pulse: `<span class="w-2 h-2 rounded-full bg-green-500 animate-pulse"></span>`
- Tech tags as pills: `<span class="px-3 py-1 text-xs font-medium rounded-full border border-neutral-800 text-neutral-400 bg-neutral-900">Tag</span>`
- Sticky navbar with blur: `backdrop-blur-md bg-brand-obsidian/80`
- Generous negative space between sections
- Card-based content with subtle hover border transitions

## Rules

- Match the reference layout closely. Don't add unsolicited features or sections
- Be specific about pixel mismatches when comparing screenshots
- Make Omar's site feel like its OWN thing — same structure, different personality
- Every design decision should feel intentional and high-end
- Accessibility: proper contrast ratios, semantic HTML, alt text, keyboard nav
